# Password Strength Meter

A C++ console application that analyzes password strength and provides humorous feedback based on the entered password.

## Features

* Detects uppercase letters
* Detects lowercase letters
* Detects digits
* Detects special characters
* Detects repeated characters
* Detects sequential characters
* Gives a strength score
* Displays funny responses

## Example

Password: Password123

Strength Score: 7/10

Feedback:

* Contains uppercase letters
* Contains lowercase letters
* Contains numbers

## Technologies Used

* C++
* Standard Library

## Download

Download the latest executable from the Releases section.

## Running
Download password_strength.exe.
Double-click the file to run it.
Enter a password when prompted.
